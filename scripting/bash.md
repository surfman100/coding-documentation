# Bash Basics

| Command | Notes |
| --- | --- |
|  \  | multiline commands | 
| single quote or none | strings |
|  >  | output to file |
| >> | append to file |
| Ctrl C | kill process |
| Ctrl L | clear screen |
| Ctrl z | suspend process |
| Ctrl D | exit shell |
| pwd | show name of directory |
